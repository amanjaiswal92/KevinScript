KevinScript
===========

Sample usage (all commands from base KevinScript directory):

    >src\main.py samples\grammar.k
    testing print statements...
    1
    True
    False
    None
    <type 'Nonetype'>
    passed.
    dict displays are not implemented yet
    all tests passed!
